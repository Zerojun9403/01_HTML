style 태그 내용 적용 확인

block 형식

        - 화면(공간) 수직 분할
        (세로 화면을 가로로 잘라서 층(행)을 나눔 )

        -한 층 (한 행 )을 모두 차지함

        -widht, height 와 같은 크기 지정 속성 사용 가능

        -block 형식의 태그 
         h1~h6(제목) p/pre(문단), hr(수평), div(영역 분할)
         시맨틱 태그

h3는 block 형식 태그 입니다.

        block 형식은 작성된 내용이 짧더라도 
        한 줄을 모두 차지하려는 성질 때문에
        다음 태그는 내용은 다음 줄에 작성됩니다. 
    
        pre 옆에 가로로 해서 이어서 작성할 수 없다는 의미로 
        태그를 닫고 열때 마다 층을 구분 지어서 생성됨

inline형식

        - 화면을 수평 분할
        (가로 화면을 세로로 잘라서 호(열)을 나눔 )

        -한 층(행)을 모두 차지하지 않음 
        -> 내부에 작성된 내용 크기 만큼 차지하려는 
        -widht,height와 같은 크기 지정 속석 사용 불가능

        -inline형식
        b(굵게),i,mark,span등

inline 형식은 행을 분할하고 한 열에서 이어서 작성되는 성질을 가지고 있어 줄이 바뀌지 않고 계속 옆으로 작성된다.
첫번째 span 두번째 span 세번째 span

inline -block형식

    -inline 특징 중 수평분할(중을 세로로 자름, 열 나눔)
    -bolck 특징 중 크기 지정 가능 장접을 합쳐 놓은 태그
    -대표적인 태그 : img태그 , iframe태그 

    iframe태그 
    -웹문서(html)내부에 또 다른 웹 문서를 추가하는 태그
